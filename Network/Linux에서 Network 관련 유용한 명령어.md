# Linux에서 Network 관련 유용한 명령어

## traceroute
어느 구간에서 느린지 확인하는 방법
ICMP 프로토콜의 `TTL`을 변경해가면서 `PING`을 보내보는 방법.
`* * *` 네트워크 디바이스에서 디바이스의 정보를 `OPEN`하지 않을 경우 나타나는 표시
특정 구간에서 느려지는 구간이 있는 경우 증빙자료로 제출 가능
```
# traceroute 15.165.230.108
```
